# IDA-9.0-RPC
Discord Rich Presence plugin for IDA 9.0

# Instructions
1. Install pypresence (pip install pypresence)
2. Drag "ida-rpc.py" over to "your_ida_installation_folder\plugins\"
3. That's it!
